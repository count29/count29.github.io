# count29-app

A single web application for [count29][0] project. Written in [VueJS][1].

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```


[0]: https://github.com/manhtai/count29
[1]: https://vuejs.org
